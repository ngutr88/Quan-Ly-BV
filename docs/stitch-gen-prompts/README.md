# Stitch Gen Prompts

Bộ prompt này chia giao diện HMS thành từng page riêng, dễ copy vào Stitch Gen và dễ quản lý kết quả sinh UI.

## Nguyên tắc chung

- Toàn bộ giao diện dùng tiếng Việt.
- Phong cách: bệnh viện hiện đại, sạch, tin cậy, dễ đọc, ưu tiên thao tác nghiệp vụ.
- Admin và Bác sĩ: desktop-first, nhiều bảng, filter, trạng thái và action rõ ràng.
- Bệnh nhân: mobile-first, thao tác ít bước, card dễ đọc.
- Không dùng dữ liệu thật. Dữ liệu mẫu nên giống bối cảnh Việt Nam.
- Trạng thái phải có text rõ ràng, không chỉ dựa vào màu.
- Các hành động nguy hiểm như hủy lịch, khóa tài khoản, hủy hóa đơn, override cảnh báo thuốc cần confirm dialog.
