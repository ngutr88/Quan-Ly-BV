# Tài liệu dự án HMS

Thư mục này dùng để quản lý tài liệu phát triển giao diện và sản phẩm cho Hệ thống Quản lý Bệnh viện (HMS).

## Tài liệu gốc

- `../PRD.md`: tài liệu mô tả sản phẩm và nghiệp vụ chính.
- `../AGENTS.md`: bộ quy tắc làm việc cho agent khi phát triển repo.

## Tài liệu giao diện

- `stitch-gen-prompts/`: bộ prompt chi tiết từng page để sử dụng với Stitch Gen.

## Cách sử dụng prompt Stitch Gen

1. Mở `stitch-gen-prompts/README.md` để xem danh sách page.
2. Chọn file page cần sinh giao diện trong `stitch-gen-prompts/pages/`.
3. Copy toàn bộ nội dung prompt trong file đó vào Stitch Gen.
4. Sinh page MVP trước, sau đó dùng các page mở rộng để hoàn thiện hệ thống.
